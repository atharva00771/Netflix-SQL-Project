# Netflix SQL Project

## 📌 Project Overview
This project analyzes the Netflix Titles dataset using MySQL. It demonstrates SQL skills through real-world business queries, including filtering, aggregation, grouping, NULL handling, window functions, and advanced SQL techniques.

---

## 🛠️ Tools Used
- MySQL
- MySQL Workbench
- GitHub

---

## 📂 Dataset
- Netflix Titles Dataset

---

## 📊 SQL Concepts Covered
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- Aggregate Functions (COUNT, MAX, MIN, AVG)
- NULL Handling (COALESCE)
- LIKE Operator
- Window Functions (ROW_NUMBER, RANK)
- Advanced SQL Queries

---

## 📈 Project Highlights
- Solved **30+ SQL queries**
- Performed data filtering and analysis
- Generated business insights using SQL
- Practiced real-world Data Analyst SQL scenarios
